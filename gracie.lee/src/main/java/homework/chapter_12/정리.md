# 12장: 스프링 데이터 JPA

## 핵심 1: JpaRepository 상속만으로 CRUD 자동 제공
- `interface MemberRepository extends JpaRepository<Member, Long>` — 구현체 없이 `save`, `findById`, `delete` 등 바로 사용 가능
- Spring이 런타임에 **프록시 객체**를 만들고, 내부적으로 `SimpleJpaRepository`에 위임

## 핵심 2: 쿼리 메서드 — 메서드 이름 = 쿼리
- `findByName(String name)` → `WHERE name = ?` 자동 생성
- `findByNameAndAgeGreaterThan` → `WHERE name = ? AND age > ?`
- 조건 2개까지는 가독성 좋음. 3개 넘으면 `@Query`로 JPQL 직접 작성이 나음

## 핵심 3: @Query
- 메서드 이름으로 표현 어려운 쿼리는 JPQL 직접 작성
- JOIN, LIKE, 서브쿼리 등에 유용
